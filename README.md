# ora-market-data-analysis
Power BI data analysis
🗂️ هيكل الـ Data Model:
• fact_Sales — 500 صفقة بيع
• dim_Products — 30 منتج في 5 تصنيفات
• dim_Customers — 100 عميل بشرائح مختلفة
• dim_Stores — 5 فروع في 3 مدن
• dim_Salesperson — 15 مندوب مبيعات

📐 مقاييس DAX اللي بنيتها:
Total Revenue | Profit Margin | Avg Order Value | YTD Sales | MoM Growth

📊 الـ Dashboard فيه:
• صفحة Overview بـ KPIs + Trends شهرية
• صفحة Customer Analysis بـ Segmentation
• Slicers ديناميكية للتصفية
